# DNA Jeopardy Generator

Standalone app for building custom Jeopardy-style boards and running a live classroom game.

## What it includes
- Board editor (categories, rows, base score)
- Question/answer authoring for every cell
- Team setup (2-5 teams)
- Game mode with turn tracking and scoring
- JSON export/import for reusing boards
- CSV template download + CSV import (`category,value,question,answer`)

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Lint
```bash
npm run lint
```
